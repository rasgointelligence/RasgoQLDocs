From your CLI or jupyter notebook, install the package from pypi:

```python
pip install rasgoql
```

From your python environment, run:

```python
import rasgoql

# Set credentials to connect to your Data Warehouse
creds = rasgoql.SnowflakeCredentials(
    account="...",
    user="...",
    password="...",
    role="...",
    warehouse="...",
    database="...",
    schema="..."
)

# Connect RQL to your Data Warehouse
rql = rasgoql.connect(creds)

# Allow rasgoQL to interact with an existing table or view in your Data Warehouse
ds = rql.dataset(fqtn='RASGO.PUBLIC.ABCD123')

# Preview the Dataset in pandas
ds.preview()

# Applying a RQL Transform to start building a SQL "chain"
chn = ds.cast(
    casts={
      'LOC_ID':'STRING',
      'DATE':'STRING'
    }
)

# Print the SQL generated by the above step
chn.sql()

# Preview the results of the SQL in pandas
chn.preview()

# Save the SQL to your Data Warehouse as a view for future consumption
new_tbl = chn.save(table_name='MY_RQL_TEST', table_type='view')
```
